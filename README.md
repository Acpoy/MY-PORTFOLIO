# Iman Asroy — Portfolio Site

## How to use
1. Open `index.html` in a browser to preview.
2. Drop your real images into `assets/` (and `assets/projects/`) using the exact filenames already referenced in the HTML — they'll swap in automatically. Until then, each spot shows a dashed placeholder with the expected filename.
3. Add your actual `cv.pdf` next to `index.html` for the "Download CV" button to work.
4. Edit any text directly inside `index.html` — the CSS in `style.css` and behavior in `script.js` don't need to change for content edits.

## File list
- index.html — all page content and structure
- style.css — visual design (dark editorial "film reel" theme, amber/coral/teal accents)
- script.js — scroll reveal animations, nav highlighting, mobile menu, cursor dot
